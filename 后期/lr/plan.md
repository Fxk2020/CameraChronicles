# Lightroom Classic 30天实战学习计划

## 📅 第1周：熟悉界面与基础操作

### Day 1-2：软件界面与导入

**今日任务：**

- [X] 打开LR，熟悉界面布局
- [X] 认识7大模块：图库、修改照片、地图、图书、幻灯片、打印、Web
- [X] 创建第一个目录（Catalog）
- [X] 导入20-30张照片（建议风景+人像混合）

**关键操作：**

- 文件 → 新建目录（选择合适位置保存）
- 导入按钮（左下角）→ 选择照片 → 选择"添加"模式
- 在图库模块查看照片

**小贴士：** 目录文件(.lrcat)要定期备份，这是你所有调整的记录！

### Day 3-4：理解直方图和基本面板

**今日任务：**

- [ ] 学会看懂直方图（Histogram）
- [ ] 理解曝光三角：高光、阴影、白色、黑色
- [ ] 练习调整5张曝光不佳的照片

**直方图解读：**

```
左侧 = 暗部信息（阴影、黑色）
中间 = 中间调
右侧 = 亮部信息（高光、白色）
堆在左边 = 曝光不足
堆在右边 = 曝光过度
```

**练习步骤：**

1. 选择一张过暗的照片
2. 提高"曝光"滑块（+0.5到+1.5）
3. 提升"阴影"恢复暗部细节
4. 降低"黑色"增加对比
5. 观察直方图变化

---

### Day 5-6：色温与色调

**今日任务：**

- [ ] 理解色温（Temperature）：蓝色←→黄色
- [ ] 理解色调（Tint）：绿色←→品红色
- [ ] 使用白平衡选择器（吸管工具）
- [ ] 调整10张色温不准确的照片

**实战练习：**

- 室内黄光照片：色温向蓝色方向移动
- 阴天偏蓝照片：色温向黄色方向移动
- 使用吸管点击应该是白色或灰色的区域

---

### Day 7：本周复习+作业

**综合练习：**
调整一组照片（10-15张），要求：

- 曝光准确（直方图分布合理）
- 色温正确（看起来自然）
- 有适当的对比度

**检查标准：**

- 照片没有死白（高光溢出）
- 照片没有死黑（阴影丢失细节）
- 肤色/天空看起来自然

## 📅 第2周：进阶调整工具

### Day 8-9：清晰度、纹理、去朦胧

**今日任务：**

- [ ] 理解三个锐化工具的区别
- [ ] 练习人像照片（降低清晰度-10到-20）
- [ ] 练习风景照片（增加清晰度+20到+40）
- [ ] 学习去朦胧工具处理雾天照片

**工具区别：**

- **清晰度**：影响中间调对比，全局效果
- **纹理**：影响细节纹理，对皮肤影响小
- **去朦胧**：专门去除灰蒙感，天空最有效

---

### Day 10-11：鲜艳度 vs 饱和度

**今日任务：**

- [ ] 理解两者区别
- [ ] 调整10张需要增强色彩的照片
- [ ] 调整5张需要减淡色彩的照片

**使用原则：**

- **鲜艳度（Vibrance）**：智能增强，对肤色保护好，首选
- **饱和度（Saturation）**：全局增强，容易过度，慎用

**人像照片：** 优先使用鲜艳度+10到+20
**风光照片：** 鲜艳度+20，饱和度+10

---

### Day 12-13：色调曲线入门

**今日任务：**

- [ ] 理解色调曲线的原理
- [ ] 练习S型曲线（增加对比）
- [ ] 学习调整单独RGB通道

**经典调整：**

```
S型曲线（胶片感）：
- 高光点向上提
- 阴影点向下压
- 中间保持不动或微调
```

**RGB通道调色：**

- 红色曲线：控制红色←→青色
- 绿色曲线：控制绿色←→品红色
- 蓝色曲线：控制蓝色←→黄色

---

### Day 14：本周复习+作业

**综合项目：**
选择一组旅行照片（15-20张），统一调整成一种风格：

- 使用色调曲线增加对比
- 适当增加清晰度或去朦胧
- 调整鲜艳度让色彩更好看
- 保持整组照片风格一致

## 📅 第3周：HSL与局部调整

### Day 15-16：HSL面板（色相/饱和度/明度）

**今日任务：**

- [ ] 理解HSL三个维度
- [ ] 练习改变天空颜色
- [ ] 练习改变植物颜色
- [ ] 学习使用目标调整工具（TAT）

**实战案例：**

**让天空更蓝：**

1. HSL → 饱和度 → 蓝色 +20到+30
2. HSL → 明度 → 蓝色 -10到-20
3. 可能需要同时调整青色

**让草地更绿：**

1. 色相 → 黄色向左（偏绿）
2. 饱和度 → 黄色和绿色 +20
3. 明度根据需要调整

**让肤色更好看：**

1. 饱和度 → 橙色 -10到-15（避免过红）
2. 明度 → 橙色 +5到+10（提亮肤色）

---

### Day 17-18：调整画笔（Adjustment Brush）

**今日任务：**

- [ ] 学会使用调整画笔工具（快捷键K）
- [ ] 练习局部提亮人脸
- [ ] 练习局部压暗背景
- [ ] 学习"自动蒙版"功能

**基础操作：**

1. 点击调整画笔图标（或按K）
2. 设置笔刷参数（大小、羽化、流量、密度）
3. 涂抹需要调整的区域
4. 调整右侧滑块

**常用技巧：**

- **提亮眼睛**：小笔刷 + 曝光+0.3~0.5 + 清晰度+20
- **美白牙齿**：曝光+0.3 + 饱和度-30
- **锐化细节**：清晰度+30 + 锐化+20

---

### Day 19-20：渐变滤镜和径向滤镜

**今日任务：**

- [ ] 学习渐变滤镜（快捷键M）
- [ ] 学习径向滤镜（快捷键Shift+M）
- [ ] 练习压暗天空
- [ ] 练习制造光线效果

**渐变滤镜案例：**

- 压暗过亮的天空：曝光-0.5到-1.5
- 提亮过暗的前景：曝光+0.5到+1.0

**径向滤镜案例：**

- 人像光晕效果：内部曝光+0.3，外部曝光-0.5
- 暗角效果：外部曝光-0.8到-1.5

## 📅 第4周：高级功能与工作流

### Day 22-23：批量处理与同步

**今日任务：**

- [ ] 学习复制和粘贴设置
- [ ] 学习同步功能
- [ ] 学习自动同步
- [ ] 批量调整一组照片（50张以上）

**操作步骤：**

1. 调整好第一张照片
2. 选中这张照片
3. Ctrl+Shift+C（Win）或 Cmd+Shift+C（Mac）复制设置
4. 选中其他照片
5. Ctrl+Shift+V（Win）或 Cmd+Shift+V（Mac）粘贴设置

**同步技巧：**

- 只同步需要的调整项（取消勾选不需要的）
- 裁剪、污点去除通常不同步
- 局部调整根据情况选择

**今日任务：**

- [ ] 创建自己的第一个预设

### Day 26-27：导出设置详解

**今日任务：**

- [ ] 理解不同导出格式
- [ ] 创建常用导出预设
- [ ] 学习批量导出
- [ ] 理解色彩空间

**导出设置建议：**

**社交媒体（Instagram/微信朋友圈）：**

- 格式：JPEG
- 质量：80-90
- 色彩空间：sRGB
- 长边：2048px
- 锐化：屏幕显示，标准

**打印输出：**

- 格式：JPEG 或 TIFF
- 质量：95-100
- 色彩空间：Adobe RGB
- 分辨率：300 PPI
- 锐化：光面纸或亚光纸

**网站使用：**

- 格式：JPEG
- 质量：75-85
- 色彩空间：sRGB
- 长边：1920-2500px

### Day 28-29：降噪与锐化

**今日任务：**

- [ ] 学习细节面板
- [ ] 练习高ISO照片降噪
- [ ] 学习输出锐化
- [ ] 理解锐化的三个参数

**降噪设置：**

- **轻微噪点（ISO 800-1600）**：明亮度30，颜色25
- **中等噪点（ISO 3200-6400）**：明亮度50，颜色50
- **严重噪点（ISO 12800+）**：明亮度70+，颜色70+

**锐化建议：**

- 数量：40-60（风光可到80）
- 半径：1.0（标准），0.8（人像），1.2（风光）
- 细节：25-40
- 蒙版：按住Alt拖动，只对边缘锐化
